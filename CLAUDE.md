# Claude Code Instructions

You are building a full production-ready MVP of a mobile application.

Project name: Death Clock AI Clone

Goal:
Create a viral mobile app that predicts a user's lifespan based on lifestyle data and presents the result in a dramatic reveal experience.

Primary Objectives

1. Build a React Native mobile app using Expo
2. Implement an engaging questionnaire experience
3. Calculate predicted lifespan
4. Display a dramatic death date reveal
5. Show a live countdown of remaining days
6. Generate shareable social cards
7. Allow optional blood test inputs
8. Deploy backend services
